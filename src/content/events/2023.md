A regional cybersecurity conference held at Virginia Western Community College.

## About
Join us for a day of learning, networking, and fun as we explore the latest trends in cybersecurity. BSides Roanoke 2023 features talks, workshops, and a Capture The Flag competition.

## Schedule

### Main Auditorium
- Opening Comments at 9:00 AM
- Talks throughout the day with various security experts
- Closing Comments at 5:30 PM

### Workshop Track
- Skills workshops located in room M318 on the 3rd floor
- Topics ranging from data intelligence to wireless security
- Lunch break included

### CTF Arena
- Beginner-friendly Capture The Flag competition
- Located in room M302 on the 3rd floor
- Runs throughout the day with demos and labs

## Speakers
- Kennedy Toomey - Security Researcher
- Logan Diomedi - Cybersecurity Specialist
- Aaron McPhall - Knowledge Management Expert
- Larry Whiteside Jr. - Security Architect
- Brian Maloney - Cloud Security Engineer
- Rob Perry - Infrastructure Security Specialist
- Jeff Hoge - Cyber Awareness Trainer

## Workshops
- Turning Data Into Intelligence / Intelligence from Data Dumps
- Unlock the Airwaves: Discovering and Decoding Wireless Messages with Software Defined Radios
- Wireless Insecurity: Breaking the Barrier to Entry
